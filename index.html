Clarence is a strong, functional language for the humanity. It was built for the advanage of fast bytecode compilation, tail-call optimization and, so much more...

Write scripts in clarence

  • Fast bytecode interactions
  • Tail-call optimizaion
  • Lazy Evaluat
  • Batteries included: send HTTP requests, read and parse JSON, work with file system, start external processes etc

Write code that expands itselfe

  • Lightweight Macros
  • Channel interactions
  • Very Readable syntax
  • Self-hosted compiler
  • Sane modularity
